# 🐱 Cat Charity Marketing Prompts — Master Guide

This folder contains 5 prompt files for Claude Code marketing skills.
All copy output must be in **Polish**. This is a Polish cat adoption charity.

---

## Files + Order of Use

```
1. product-marketing-context.md   ← START HERE. Run first, always.
2. copywriting.md                 ← All page copy (Polish)
3. seo-strategy.md                ← SEO, keywords, schema, AI search
4. cro-architecture-psychology.md ← Site structure, CRO, colour, vibe
5. social-content-strategy.md     ← Instagram, Facebook, blog, emails
```

---

## How to use in Claude Code

### Step 1 — Fill in the blanks in product-marketing-context.md
Before running anything, open `product-marketing-context.md` and fill in:
- Nazwa fundacji / schroniska
- Dokładny adres / miasto
- Rok założenia
- KRS number

### Step 2 — Run product-marketing-context first
In Claude Code terminal:
```
> Read product-marketing-context.md and remember this as the foundation for all marketing work on this project
```

### Step 3 — Run specific skills
```
> Using the copywriting.md prompt and our product marketing context, write all homepage copy in Polish

> Using seo-strategy.md, produce a complete keyword map for Polish cat adoption searches

> Using cro-architecture-psychology.md, design the full site architecture and navigation in Polish

> Using social-content-strategy.md, create a 30-day Instagram content calendar with captions in Polish
```

### Step 4 — Combine for specific tasks
```
> Using copywriting.md and marketing-psychology from cro-architecture-psychology.md, write the /wspieraj donation page — focus especially on the 1.5% podatku section

> Using seo-strategy.md Part 5 and copywriting.md, write the cornerstone blog post "Jak adoptować kota w Polsce"
```

---

## Key things Claude must always remember

1. **Polish language only** for all copy, page text, cat profiles, social content, emails
2. **Never guilt-trip** — warm, honest, direct tone always
3. **1.5% podatku** is the highest-priority, lowest-friction donation CTA — always include it
4. **Individual cat names + photos** beat generic charity imagery — always personalise
5. **The audience fears being judged/rejected** — always address this in adoption process copy
6. **Foster homes (domy tymczasowe)** are undersold — always include as secondary CTA
7. **Honest cat profiles** are a key differentiator — never oversell a cat's personality
